CHATTING RULES

1. No Pinging Content Creators: If You Ping Any Content Creator (Like @ItsMarwan) Without Reason, You'll Get Muted For 3 Days.
2. No Swearing/Bad Language: Never Swear, Use Slurs, Be Toxic, or Say Anything Racist. Respect Everyone In The Server.
3. No Self-Promotions: Only Send Your Maps In The ⁠your-builds Channel if youre caught Promoting You'll Get Muted For 5 Days.
4. Listen To Moderators & @ItsMarwan: Mods Are Here To Help. Respect Their Decisions Or You Might Get Muted.
5. Helping Others: Don’t Say “idk” Or “help me pls” Without Trying. If You Can’t Help, Stay Quiet. Breaking This Rule = 1 Week Mute.
6. No Politics Or Religion: These Topics Aren’t Allowed. If You Start Such Discussions, You’ll Be Banned For 3 Weeks.
7. Be Mature & Respectful: No Drama, No Toxic Replies, No Picking Fights. This Is A Chill Place.
8. No Mini-Modding: Do Not Act Like A Mod Unless You Are One. Let The Staff Do Their Job.
9. No Begging: Don’t Beg For Roles, Ranks, Free Stuff, Or Special Permissions.
10. Respect Other Cultures: Don’t Mock, Joke About, Or Disrespect Anyone’s Country, Accent, Religion, Or Background.

CHANNEL USAGE

1. Support Channels: Only Ask For Help In ⁠community Or ⁠ask-marwan. Not ⁠general Or Off-Topic Channels.
2. Channel Usage: Use Each Channel For Its Intended Purpose. Read The Description Before Typing.
3. Usage Of ⁠ask-marwan: This Is Only For Direct Questions To @ItsMarwan. Don’t Use ⁠general To Ask Him Anything.
4. Follow Forums Rules: Some Forums Have Rules Or Templates. Read Them Before Posting.
5. Media Channels: Don’t Spam Random Images, Videos, Or Memes. Only Post Relevant Stuff To The Topic.
6. English Only: This Server Uses English. You Can Chat In Other Languages In Designated Channels If Available.

KEEPING SERVER SAFE

1. Dealing With Bugs: Found A Bug Or Server Issue? DM @ItsMarwan With Details So It Can Be Fixed.
2. Reporting Users: Tag @report_user And Explain The Issue Clearly. Misusing This Gets You A 2 Day Mute.
3. Discord TOS: Always Follow Discord’s Terms Of Service. Breaking Them Can Get You Banned From This Server & Discord Itself.
4. Suspicious Activity: Don’t Share Scam Links, Phishing, Or Fake Giveaways. Instant Ban If Caught.
5. Hacked Accounts: If Your Account Was Hacked Or Acts Suspiciously, You Will Be Temporarily Banned Until Verified.
6. Privacy: Don’t Share Private DMs, Personal Info, Or Other People’s Content Without Permission.

CHANNEL RESTRICTIONS

1. Links: Only Allowed In ⁠your-builds And ⁠promotions. Posting Links Elsewhere = Warning + Link Deleted.
2. Spam Messages: No Spamming Texts, Emojis, Images, Or Repeated Messages.
3. Server Invites: Don’t Send Invites To Other Servers Unless It’s In ⁠promotions.
4. NSFW/Offensive Content: This Is A Family-Friendly Server. No NSFW Content Anywhere. Doing So = Instant Ban.
5. Impersonation: Don’t Pretend To Be Staff, Creators, Or Other Members. This Gets You Banned.
6. AI & Bot Use: Only Use Bots In The Bot Channels. Don’t Spam Bot Commands In General Chat.

EVENT RULES

1. No Cheating: If You’re Caught Cheating Or Breaking Event Rules, You’ll Be Disqualified And Possibly Banned.
2. Respect Event Hosts: If You're Participating In A Tournament, Contest, Or Giveaway, Follow The Host’s Instructions.
3. Don’t Ask For Wins: Asking For Free Wins, Roles, Or Giveaway Rerolls Will Get You Disqualified.
4. One Account Only: Do Not Join Events Using Alts. Only Your Main Account Is Allowed.
